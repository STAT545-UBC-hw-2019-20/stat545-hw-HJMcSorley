## Assignment 04: Tidy data and joins

The focus of this assignment is to practice problem solving in R using two key components of data wrangling: data aggregation and reshaping. 

Key functions used were:
  * pivot_wider()
  * pivot_longer()
  * left_join()
  * anti_join()
  * full_join()
